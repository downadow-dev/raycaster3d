# raycaster3d

Сетевая 2.5D-игра на основе метода бросания лучей (англ. _ray casting_).

## Информация

Данный проект сгенерирован с помощью [gdx-liftoff](https://github.com/libgdx/gdx-liftoff) версии `1.13.5.1`.
