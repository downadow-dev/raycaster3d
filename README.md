# raycaster3d

Сетевая 2.5D-игра на основе метода бросания лучей (англ. _ray casting_).

## Создание сервера

Для того, чтобы создать сетевую игру, введите в командной строке:
```
php -S 0.0.0.0:8000
```
Запустите игру, введите HTTP-адрес в формате `http://<ip>:<port>`,
нажмите `[server]`. Теперь другой человек сможет подключиться к вам.
Узнать свой IP-адрес в сети можно, запустив следующее:
```
ip addr
```
